# Computação Paralela

No âmbito da UC de Computação Paralela, foi-nos proposta a otimização de um código que simula o movimento de partículas ao longo do tempo, dividida em 3 fases:
- Instruction Level Parallelism (ILP)
- OpenMP
- CUDA
